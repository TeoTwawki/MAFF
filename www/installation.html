Installation

Visit the Firefox Add-ons website to install Mozilla Archive Format in Firefox or SeaMonkey.

You can also install from the Beta Channel to get new versions as soon as they are available.

Compatibility

The Mozilla Archive Format add-on can be installed on the latest available versions of Firefox, Firefox Extended Support Release (ESR), and SeaMonkey. The most recent version of the add-on is not expected to be compatible with older versions of the browser or with pre-release channels such as Nightly and Beta.
